# CG-Project

Voxel Landscape

## Build and run

    chmod +x INSTALL
    ./INSTALL

If using Windows, install [Git for Windows](https://gitforwindows.org/), choose 'Git Bash here'
in the context menu of the git folder, then run the commands.

## Dependencies

cmake, libgl 2.1+, glfw3, magnum, corrade
